# Gulp

Gulp 는 자주 사용 되는 기본 환경을 빠르고 간편하게 만들 수 있게 도와주는 도구이다.
이는 반복적인 작업을 줄일 수 있게 해주며, 이로 인해 생산적인 일에 좀 더 집중할 수 있게 해준다.
프론트 엔드도 점점 복집해지는 추세이기 때문에 요즘은 자동화 도구가 필수라고 할 수 있습니다.

Gulp를 통해서 이용 할 수 있는 대표적인 주요 기능으로는 다음와 같습니다.

  • 웹서버를 띄울 수 있습니다.
  • 파일 저장시 브라우저의 자동적인 리로딩을 가능하게 해줍니다.
  • Sass / LESS 같은 간편한 문법을 사용하지만, 자동 빌드를 해준다.
  • images 파일 등 파일의 용량을 줄여주고, css는 압축이 가능하는등 설정에 따라 최적화가 가능합니다.

# Gulp 시작해보기

! 시작 전 노드설치는 필수 [ Node.js ] gulp는 노드 기반이기 때문에 node가 설치되어 있다는 가정하에 진행합니다. npm 을 통해서 -g 깃발을 표시할 경우 글로벌 영역에 설치가 되어서, 전 영역에서 사용이 가능합니다.

npm install gulp -g

프로젝트를 실행할 폴더를 만들고, 디렉토리 안에서 npm init을 통해 package.json를 만듭니다.
이는 프로젝트에 어떤 정보가 담겨있는지 확인이 가능하고, 설치시 --save-dev를 통해 기입이 됩니다.

package.json과 node_modules폴더가 준비 되었다면, 앞으로 진행 할 프로젝트의 파일 구조를 지정해 줘야 합니다.
기본적인 트리구조를 사전에 정리함으로써, minify 된 파일을 어디에 저장 할지 디렉토리를 지정하며, gulp 세팅을 해줘야 한다.

# 폴더 구조 정리하기

|- app/
    |- css/
    |- fonts/
    |- images/
    |- index.html
    |- js/
    |- scss/
|- dist/
|- gulpfile.js
|- node_modules/
|- package.json

app 폴더에서 만들어서, dist 폴더에 최적화된 파일만 저장하도록 했습니다.
현재 보이지 않는 디렉토리들은 구조화 하면서 직접 만들어 주도록 하겠습니다.
gulpfile.js 파일은 어떻게 Gulp를 세팅할지 가이드가 되는 파일이 될겁니다.

# Gulp 설정하기

먼저 Gulp를 불러옵니다.

var gulp = require('gulp');

걸프는 task-name 에 적합한 이름을 설정해 준 후, 함수 안에 실행할 내용을 적어주시면 됩니다.

gulp.task('task-name', function() {
    // Stuff here
});

그러면 "hello"라는 task-name을 선언하고, "Hello world"를 찍어보도록 할게요. 명령어는 gulp + task-name 으로, gulp hello 라고 호출했습니다.

gulp.task('hello', function() {
    console.log('Hello world');
});

실행되는걸 확인해보실수 있습니다.

이런 식으로 gulp 세팅을 해봤는데, gulpfile.js에 원하시는대로 커스터마이징을 해두면, 원하는 폴더에 원하는 방식으로 파일을 최적화 하면서 작업할수 있습니다. 그래서 기본 세팅이 중요합니다. 물론, 한번에 설치할수 있도록 패키지 되어있는 방식도 있습니다.

# Gulp 시작해보기

Gulp.js 사이트에 접속하면, 실제 설치해서 적용 가능한 다양한 플러그인들을 검색해 보실 수 있지만, 우선은 보편적으로 많이 사용하시는 몇가지 플러그인들을 설치해 보겠습니다.

Gulp-sass 를 먼저 설치합니다. Sass를 css로 컴파일 해주는 플러그인 인데, 커멘드창에 npm-install 을 통해서 설치가 가능합니다.
설치할때 --save-dev를 같이 입력해 주시면, package.json 파일에 함께 입력된다는 것이 포인트입니다. (코드 앞에 입력되는 $는 디렉토리를 나타내므로 제외시키고, 입력해주세요.)

<p>$ npm install gulp-sass --save-dev</p>

이제 gulpfile.js 파일에 일전에 설정하기에서 배운 방식과 동일하게, gulp-sass를 사용해 볼게요.

var gulp = require('gulp');
// Requires the gulp-sass plugin
var sass = require('gulp-sass');

gulp.task('sass', function(){
    return gulp.src('app/scss/styles.scss')
        .pipe(sass()) // Converts Sass to CSS with gulp-sass
        .pipe(gulp.dest('app/css'))
});

6번 줄에 보이는 src는 폴더 디렉토리 구조에 따라 변경이 가능합니다.
불러온 scss 파일을 scss()로 css 파일로 변환 시켜준 후에, dest('원하는 디렉토리')에 다시 저장을 시킵니다.

원하는 .scss 파일을 만들고, 해당 명령어를 사용하고 싶다면 저희는 간단하게 gulp sass 라는 예약어를 사용하면 됩니다. 물론 pipe를 사용해서 css를 추가 minify 적용해도 됩니다.

$ gulp sass
